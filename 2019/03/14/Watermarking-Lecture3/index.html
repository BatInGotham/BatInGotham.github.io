Lecture 3: Models of Watermarking

Components of Communications Systems

  • x is signal can be transmitted over the channel, but m is not.
    • Source coder: draw symbols in some alphabet.
    • Modulator: converts a sequence of symbols into a physical signal.
  • Transmission in channel add noise n.

Input message —(m)–> Channel encoder —x–> (+Noise) —(y)–> Channel decoder —(m_n)–> Output message

Classes of Transmission Channels

According to the type of noise function

  • Additive noise y=x+n
  • Fading noise y=v[t]x+n

Secure Transmission

  • Message layer: cryptography, preventing unauthorized read/write

  • Transport layer: spread spectrum communication

Blind Embedding (E_BLIND)

One bit only message m = 0|1

  • Encoding into to message pattern: wm=(2m-1)wr.
  • Modulate to added pattern: wa=a*wm.
  • Embedding: cw=co+wa.
