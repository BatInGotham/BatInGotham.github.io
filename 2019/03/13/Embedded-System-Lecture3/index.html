Lecture 3: 认识CPU

嵌入式CPU——MCU

MCU：一定含有外设，且至少包含定时器和GPIO，一般还具有ADC、UART和PWM等。

CPU：只有指令译码部分，不含有任何外设，如x86

SoC：带有存储器及某个特定外设的MCU（如esp8266）

嵌入式CPU分类

  • 8 bit vs 32 bit
  • CISC vs RISC
  • not ARM vs ARM
  • without MMU vs with MMU

下载方式

  • 片上存储器
  • 外接存储器

功能

  • GPIO：数量、驱动能力
  • UART：数量
  • AD：分辨率、速度、数量
  • 定时器：数量、长度
  • SPI：速度
  • I2C：速度
  • 其他通信：CAN、USB

封装

BGA：体积小、价格低、加工成本高

QFN：手工焊接难

QFP：加工成本低、适合手工焊接

DIP：体积大、价格高
