2019/03/04 Operating System

1. 操作系统是一种（ ）

A. 通用软件

B. 系统软件

C. 应用软件

D. 软件包

答案选B，操作系统是计算机系统中最基本的系统软件

操作系统是指控制和管理整个计算机系统的硬件与软件资源，合理地组织、调度计算机的工作与资源的分配，进而为用户和其他软件提供方便接口与环境的程序集合。

2. 操作系统是对（ ）进行管理的软件

A. 软件

B. 硬件

C. 计算机资源

D. 应用程序

答案选C，操作系统管理计算机的硬件和软件资源，统称为计算机资源

3. 下面（ ）资源不是操作系统应该管理的

A. CPU

B. 内存

C. 外存

D. 源程序

答案选D，源程序是面向编程人员的，不是操作系统管理的内容

误区：操作系统管理文件，为何不管理源程序代码文件？

操作系统关心的是文件的逻辑结构、物理结构、文件内部结构、多文件之间如何组织的问题，而不关心文件的具体内容。源程序此处的意义不是指文件

4. 现代操作系统中最基本的两个特征是（ ）和（ ）

答案是并发和共享

现代操作系统的特征有四个：并发Concurrence、共享Sharing、虚拟Virtual、异步Asynchronism。其中前两个为基本特征

5. 操作系统与用户通信接口通常不包括（ ）

A. Shell

B. 命令解释器

C. 广义指令

D. 缓存管理命令

答案选D，Shell是命令解析器，属于命令接口；命令解释器属于命令接口；广义指令即系统调用。

系统中的缓存全部由操作系统管理，对用户透明，操作系统不提供管理系统缓存的系统调用。

6. 下列选项中，不属于多道程序设计的基本特征的是（ ）

A. 制约性

B. 间断性

C. 顺序性

D. 共享性

答案选C，引入多道程序后，程序的执行就失去了顺序性与密闭性。顺序性是单道程序的基本特征。

7. 计算机开机后，操作系统最终被加载到（ ）

答案是RAM

操作系统的程序会被自动加载到内存的系统区，这段区域是RAM

总结

今天学习的部分为操作系统的基本概念（OS-1.1）。

操作系统的定义是指控制和管理整个计算机系统的硬件和软件资源，合理地组织、调度计算机的工作与资源的分配，进而为用户和其他软件提供方便接口与环境的程序集合。

操作系统是计算机系统中最基本的系统软件。

操作系统的四个特征为并发、共享、虚拟、异步，其中并发和共享是其基本特征。

操作系统的功能分为三部分：作为计算机系统资源的管理者；为用户和计算机硬件提供接口；用于扩充机器。

  • 作为计算机系统的管理者，操作系统主要进行处理器管理、存储器管理、文件管理、设备管理
  • 提供接口，主要分为命令接口（联机/脱机）和程序调用两部分
  • 扩充机器，指的是操作系统在裸机之上，构建了扩充机器（或称为虚拟机）
